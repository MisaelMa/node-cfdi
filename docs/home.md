## @cfdi/xml

Este modulo genera un CFDI a partir de clases TypeScript lo que facilita la creacion de XML y sellarlo sin ningun problema de compatibilidad.

- Soporte completo para CFDI 4.0
- Complementos: Pagos 2.0, Carta Porte 2.0, INE 1.1, Aerolineas, IEDU
- Generacion de cadena original sin Java (via `@cfdi/transform`)
- Certificados de Sello Digital con `@cfdi/csd` (Certificate, PrivateKey, Credential)
- Validacion contra esquemas XSD del SAT
