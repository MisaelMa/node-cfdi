# @cfdi/server

Aplicacion web Next.js para el ecosistema CFDI. Proporciona una interfaz de usuario para la generacion y gestion de comprobantes fiscales.

## Desarrollo

```bash
pnpm dev
```

Abre http://localhost:3000 en tu navegador.

## Build

```bash
pnpm build
```
